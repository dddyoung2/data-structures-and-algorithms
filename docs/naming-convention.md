# Naming Convention

## English Version

### General Rules
- Use underscores instead of spaces.

### Implementation file

#### Format:
algorithm_or_data_structure.py

#### Examples:
- bfs.py
- dfs.py
- binary_search.py
- union_find.py
- bst.py

### Problem file

#### Format:
platform_problemNumber_shortTitle.py

#### Examples:
- leetcode_1971_valid_path.py
- baekjoon_1260_dfs_bfs.py
- programmers_network.py
